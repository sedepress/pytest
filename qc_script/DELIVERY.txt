📊 API 接口

GET /api/hot-say/list
  参数: page, page_size, keyword, type, source, sort_key, sort_dir
  返回: 列表数据 + 分页信息

GET /api/hot-say/stats
  参数: type, source
  返回: 统计数据

POST /api/hot-say/update-score
  Body: {"id": 123, "score": "A"}
  返回: 操作结果

POST /api/hot-say/delete
  Body: {"id": 123}
  返回: 操作结果
